Cocktail

Höchste Priorität hat bei uns der Wohlfühlfaktor. Kommt also nur in Kleidung in der Ihr euch wohlfühlt! Wer dennoch eine kleine Richtlinie haben möchte, findet hier einen kleinen Vorschlag.

Frauen tragen ein sommerliches Kleid, einen eleganten Einteiler oder ähnliches.
Männer kommen im stilvollen Anzug, wer will trägt dazu Krawatte oder Fliege. Eine legere Alternative wäre eine elegante Chinohose mit Hemd.
